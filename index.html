What is the difference between Page Views, Page Visits and Unique Visitors of a web page? How would you count them in order to get a reliable result?

Answer: A page view is when a visitor navigates to a new page. A page visit is a series of views in one session. A session is over after a certain period of viewer inactivity, after that the next page view counts as a new visit. The number of unique visitors is like the number of visits, but multiple visits from the same computer still counts as one unique visitor.
